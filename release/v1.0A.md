# High Dice v1.0 Alpha Release Notes

Welcome to the first alpha release of High Dice!

## Highlights

- **Core gameplay:** Play poker-inspired hands with 5 dice.
- **Reroll system:** Select dice to reroll and improve your hand.
- **Leveling:** Consume stakes to level up and earn upgrade points.
- **Hand upgrades:** Boost multipliers for specific hands.
- **Responsive UI:** Simple controls and console log feedback.
- **Game over & reset:** Restart anytime from the UI.

## Known Issues & Limitations

- Some features and polish are in progress.
- No persistent high score or ranking yet.
- Mobile UI need improvements.
